## The problem

Given a CSV of runner telemetry (step frequency, heart rate, distance, height), perform two tasks:

- **Task 1:** compute step frequency per minute and report the average.
- **Task 2:** run K-means clustering on height and heart-rate data; visualize with a scatter plot.

Full problem statement in [`tasks.pdf`](./tasks.pdf); source data in [`data.csv`](./data.csv).

## The meta-story
